Instructions to download the required jar to send emails.
Go to: 
https://javaee.github.io/javamail/
Download by clicking "javax.mail.jar" about half way down the page
Then put the jar where ever you have your other jars
Right click the CSC3610_Group_Project project, go to properties
Go to Java Build Path
Select Libraries
Then click Add External Jars
Then select the javax.mail jar
Hit apply then OK
